### Instalación de paquetes
Para correr los diferentes programas se deberán instalar previamente los siquientes paquetes:

* numpy
`pip install numpy`
* matpotlib
`pip install matpotlib`
* cv2
`pip install opencv-contrib-python`
* torch
`pip install torch`
* mediapipe
`pip install mediapipe`
